# DS Consultoría — Panel de Control

## Qué es
Panel de gestión para Diego Selem, consultor estratégico-operativo de PyMEs en Argentina. Permite trackear el estado de todos sus clientes activos, sus tareas pendientes, las tareas asignadas a cada cliente, y las iniciativas en curso por proyecto.

## Usuarios
Un solo usuario: Diego Selem. Consultor independiente que accede desde Mac y celular.

## Funcionalidades principales
- Vista general de todos los clientes con estado, tareas pendientes y próxima reunión
- Vista detalle de cada cliente con tareas editables (mías y del cliente) e iniciativas con estado
- Actualización de reunión via texto libre procesado por IA que extrae tareas automáticamente
- Edición inline de tareas e iniciativas
- Agregar y eliminar tareas e iniciativas manualmente

## Tono visual
Profesional, oscuro, moderno. No corporativo genérico. Debe sentirse como una herramienta propia, no un template.

## Stack
React + Vite. Sin librerías de UI externas. CSS inline con variables.
